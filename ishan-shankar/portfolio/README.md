# Ishan Shankar — Portfolio Website

A clean, responsive personal portfolio website built with plain HTML, CSS, and JavaScript as a Week 1 task submission.

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with name, role, and profile avatar |
| About | `about.html` | Personal bio/introduction section |
| Contact | `contact.html` | Contact form for hiring/enquiries |

## Tech Stack

- **HTML5** — semantic markup across 3 pages
- **CSS3** — custom styles with flexbox layout and smooth transitions
- **Font Awesome** — social media icons (Facebook, WhatsApp, LinkedIn, X/Twitter)
- **Google Fonts** — [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) typeface

## Features

- Responsive navbar with active page highlighting and a **Hire Me!** CTA button
- Profile avatar displayed on the homepage (`avatar.svg`)
- Contact form with name, email, and message fields
- Fixed footer with social media icon links
- Fully responsive design with breakpoints at **1024px**, **768px**, and **650px**
- Subtle hover animations on buttons, nav links, and social icons

## Project Structure

```
portfolio/
├── index.html      # Home page
├── about.html      # About page
├── contact.html    # Contact page
├── style.css       # Global stylesheet
└── avatar.svg      # Profile image
```

## Getting Started

No build tools or dependencies required. Just open `index.html` in your browser:

```bash
# Clone the repo
git clone https://github.com/shnkr-ishan/Week-1-Task-Submission.git

# Navigate to the portfolio folder
cd Week-1-Task-Submission/ishan-shankar/portfolio

# Open in browser
open index.html
```
